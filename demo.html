DEMO Avanzar el tiempo:

Máquina de Encuestas

Demo interactiva sin backend: los datos viven en esta pestaña y se reinician al recargar. En el producto real, el cierre lo dispara el sistema del operario vía API, el email sale solo y WhatsApp funciona con un tap (link wa.me, sin API de Meta). Usá “Avanzar el tiempo” para ver el envío diferido (45 min), el reenvío automático (48 hs) y el seguimiento semanal de casos.
